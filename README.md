LOCALOS
LOCALOS — это мощная и легковесная платформа для системного администрирования прямо в вашем терминале. Инструмент написан на языке Python и спроектирован как модульная система: вы можете управлять сервером, контейнерами, сетями и безопасностью, не покидая консоль.

🚀 Основные возможности
Система построена по модульному принципу. В данный момент доступны:

System Information — оперативный мониторинг состояния системы.

Process Manager — контроль запущенных процессов и потребления ресурсов.

Network Tools — диагностика сетевых интерфейсов и соединений.

Crypto & Password Vault — встроенные инструменты шифрования и безопасное хранилище паролей.

Docker Manager — полноценное управление контейнерами, сетями и томами (реал-тайм статистика и логи).

Log Analyzer — удобная работа с системными логами (journald, apt, postgresql и др.) с возможностью поиска и фильтрации.

Task Scheduler — управление запланированными задачами.

Firewall — управление правилами сетевой безопасности.

🛠 Установка
Для работы требуется установленный Python 3.x.

Bash
# Клонирование репозитория
git clone https://github.com/Anonimman01/LOCAL-OS-CLI.git
cd LOCAL-OS-CLI

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python main.py
⚡ Преимущества архитектуры
Модульность: Каждый инструмент — это независимый модуль.

Lazy Loading: Тяжелые библиотеки загружаются только тогда, когда вы обращаетесь к конкретному модулю. Это сохраняет минимальное потребление оперативной памяти.

TUI (Text User Interface): Интуитивно понятный интерфейс на базе библиотеки rich, обеспечивающий скорость работы даже на удаленных серверах через SSH.
